# Premium rendering changes - v3

This update adds the latest rendering refinements observed on short SSN executive decks.

## Main changes

- Added cover restraint rule to avoid large decorative accent lines.
- Added process scale rule so process/timeline slides use more of the safe area and do not look too small.
- Added wide process row helper with larger cards, stronger titles and visible connectors.
- Added final slide dominance rule: the final slide must choose model, next steps, or decision request as the primary purpose.
- Strengthened QA to flag small process diagrams, heavy cover accents and final slides with two equal visual systems.
- Updated root `SKILL.md` with latest rendering refinements and review checks.
